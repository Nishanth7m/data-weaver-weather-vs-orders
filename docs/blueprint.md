# **App Name**: Weatherly Orders

## Core Features:

- CSV Data Loading: Load order data from the bundled CSV file.
- Firestore Integration: Optionally read data from Firestore to display orders.
- City & Date Filtering: Filter order data based on selected city and date range.
- Deterministic Temp Simulation: Simulate temperature based on city and date without external API calls. Weather Tool determines correlation of temperature and orders.
- Dual-Axis Chart: Plot orders and simulated temperature on a dual-axis chart.
- Order Data Display: Display the data with Date and Orders columns in an HTML table

## Style Guidelines:

- Primary color: Muted blue (#6699CC), for a calm, data-focused feel, inspired by weather maps.
- Background color: Light gray (#F0F0F0), offering a clean and neutral backdrop.
- Accent color: Soft green (#8FBC8F), complementing the blue with an analogous feel.
- Body and headline font: 'Inter' for a modern, neutral, and readable interface.
- Simple, outline-style icons for weather and data representation.
- Clean, well-spaced layout with clear sections for filters, data table, and chart.
- Subtle transitions and animations on data updates and filter changes.
